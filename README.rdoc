== AlwaysNote

== Demo Information

To try out a live demonstration of this project, visit http://alwaysnote.herokuapp.com
and log in as the user "Demo" with the password "password".


== About this project / Features

AlwaysNote is a web-based clone of the Evernote desktop app. It allows any user to 
create a notebook and start writing rich-text notes that be shared with other users. 
The back-end is built in Rails, whie the front end is done in backbone.js, HTML, and CSS.
